### 简介
    <p>资源来自 <a href="https://forum.freemdict.com/t/topic/42230">freemdict这个帖子</a>，我主要做了以下工作：
    <ul>
    <li>详细校对工作。</li>
    <li>移除了个别人名的案例，在我看来这些个别案例不过是一些俗名，好比小苏打之于碳酸氢钠，小苏打这个名字没有必要专门去说明，所谓推荐讨论的是化合物的系统命名。对于英语姓名原则上字母排列一样，读音发音一样，那么在中文里面最好都规范对应读作某一个名字。如果翻译工作人员在遇到陌生的外国姓名无从下手，可以参考本工具提供的推荐建议，这里说是推荐的意思就是说翻译人员当然可以不遵循推荐，而有自己的小心思小技巧，但极个别的案例不影响这里的总体讨论。</li>
    <li>提供原书的markdown格式，Github项目地址在这里，本工具就是基于这个文本建立的一个搜索工具。</li>
    </ul>
    </p>
